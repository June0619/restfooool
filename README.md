## Goals
  - RESTful 설계 이해 
    1. 자원 주체 URI 설계
    2. Http Status 응답 코드
    
  - 구조
    1. POST
    2. MEMBER
  - template 페이지 따로 만들지 않고 서버만 작성 예정
  - 커밋 규칙 지키기
    1. 작업 단위 별 커밋 (뭉탱이로 하지 않기)
    2. 에러를 유발할 수 있는 소스는 최대한 커밋 자제
